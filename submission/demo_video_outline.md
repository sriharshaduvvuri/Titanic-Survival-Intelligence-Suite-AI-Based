# Demonstration Video Outline - Titanic Survival Intelligence Suite

This outline represents the storyboard and narration outline for a 3-minute video showcase of the Titanic Survival Intelligence Suite.

## Section 1: Introduction & Landing Page (0:00 - 0:45)
- **Visuals**:
  - Landing page viewport scrolling down smoothly.
  - Hovering over floating glassmorphic diagnostic stats cards.
- **Narrator Voiceover**:
  - *"Welcome to the Titanic Survival Intelligence Suite, a full-stack predictive machine learning application designed to assess passenger survival probabilities."*
  - *"Built using React, TypeScript, Tailwind, and FastAPI, this SaaS-style dashboard provides dual-classifier evaluations using Random Forest and XGBoost."*

## Section 2: Registration & Dashboard Analytics (0:45 - 1:30)
- **Visuals**:
  - Filling in user credentials in the Registration form.
  - Accessing the main Dashboard showing interactive charts (survival by gender, class pie chart, age demographics stacked bar).
- **Narrator Voiceover**:
  - *"New users sign in securely via JWT. The main analytics panel displays real-time demographic distributions built with Recharts, with smooth slide-in hover tooltips."*

## Section 3: Predictive Analytics & XAI (1:30 - 2:30)
- **Visuals**:
  - Adjusting sliders and selectors in the Prediction Center and clicking evaluate.
  - Reviewing the survival probability gauge (speedometer ring).
  - Navigating to the Explainable AI panel and inspecting the horizontal red/green SHAP force contributions.
- **Narrator Voiceover**:
  - *"In the Prediction Center, adjustments to age, class, and fare yield comparative RF and XGBoost scores. We approximate local SHAP values to explain individual passenger results, mapping positive and negative forces."*

## Section 4: CSV Processing, PDF Reports, & Admin Logs (2:30 - 3:00)
- **Visuals**:
  - Dragging a sample CSV onto the batch predictions area.
  - Reviewing results, downloading analytical PDFs, and checking admin audit logs tables.
- **Narrator Voiceover**:
  - *"The suite handles batch predictions with file size checks, exports Helvetica analytical PDFs, and audits activity logs. Thank you for reviewing this submission."*
